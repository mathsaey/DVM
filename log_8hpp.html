Detailed Description

DVM Log abstraction layer.

Author
Mathijs Saey

This header provides all the required function to create and use logs. This is done by creating a few light wrappers around boost/log.

Any logging configuration should happen in LOG_SETUP()

Logging in DVM can be done in the following way:

  • Creating a log:
    Log log = LOG_CREATE("channel");
  • writing to a log:
    LOG(log, level) << "message";
