Detailed Description

This class represents a node in a dataflow graph. A node represents a single operation on an arbitrary amount of input data.

This input data is provided by other elements in the form of a Token. Once the node received all of the input tokens it needs, it executes it's operation and sends the results to other nodes that are waiting for this data.
