Detailed Description

Contains boost log configuration.

Author
Mathijs Saey

This file imports the various headers for boost/log. It also declares the the severity levels and type necessary to create logger instances

In order to create a global logger, use the following code.

* BOOST_LOG_INLINE_GLOBAL_LOGGER_INIT(loggerName, DVMLogger) {
* return DVMLogger(boost::log::keywords::channel = "channel name");
* }

Using a logger can be done with the following macro

* BOOST_LOG_SEV(loggerName::get(), info) << "Hello, world!";
