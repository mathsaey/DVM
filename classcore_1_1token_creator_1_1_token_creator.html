Detailed Description

DVM Token creator.

The token creator is responsible for wrapping results of instructions in tokens.

def core.tokenCreator.TokenCreator.changeContext (   self,
  token,
  inst 
)

Change the context of a token.

This will bind the context of this token to a certain return instruction.

The runtime may decide to send these tokens to a different runtime core.

Multiple tokens that are sent from the same instruction with the same context will receive the same context.

Parameters
tokenThe token to modify. The destination port of this token will remain unchanged.
instThe instruction that called the contextchange. this has to be a contextChange instruction.
