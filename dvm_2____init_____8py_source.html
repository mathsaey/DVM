43 # Create an instruction and add it to memory.
44 #
45 # \param constructor
46 # The instruction constructor
47 # \param argLst
48 # The arguments to pass to the constructor.
49 # The exact sructure of this list depends on the
50 # instruction type.

56  inst = constructor(*argLst)
57  return memory.add(inst)
