Detailed Description

Split instruction.

This instruction only accepts compound data. It stores a destination sink, a return sink and a merge list.

When it receives a compound data type, it splits up the type into it's elements. For each of those elements, it sends the element to idx 0 of the destSink. The index of this element is sent to port 1. The index can be used to reconstruct the compound type later on.

Finally, the split instruction also accepts a merge list. The merge list is a list of addresses. On splitting a compound data type, the split instruction will tell the context matcher to reserve the correct amount of slots for the length of the compound data type.
